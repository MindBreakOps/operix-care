## <img src="https://api.iconify.design/lucide/shield-check.svg?color=%2310b981" width="24" align="top" /> Enterprise Security & Compliance

We are committed to strict data protection. OPERIX Care is designed in compliance with **Saudi CCHI regulations** and international **HIPAA standards**. 

* <img src="https://api.iconify.design/lucide/lock.svg?color=%2364748b" width="16" align="top" /> **Tenant Isolation:** Unbreakable PostgreSQL Row Level Security (RLS) ensures no data leakage between facilities.
* <img src="https://api.iconify.design/lucide/server.svg?color=%2364748b" width="16" align="top" /> **Data Residency:** All patient data is encrypted and stored securely in certified data centers within the Kingdom of Saudi Arabia.
* <img src="https://api.iconify.design/lucide/fingerprint.svg?color=%2364748b" width="16" align="top" /> **Auth:** Passwordless OTP, Google Enterprise integration, and robust session fingerprinting.
